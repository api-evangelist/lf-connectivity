# LF Connectivity (lf-connectivity)
LF Connectivity is a Linux Foundation project umbrella launched with Meta to improve enhanced access to networks. It includes sub-projects like Magma for mobile network core, Terragraph for wireless backhaul, and ISP Toolbox for network planning and analysis.
